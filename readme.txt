A simple yet powerful WordPress resource monitor to track memory, queries, plugin usage, and user sessions.

== Description ==
Site Task Manager gives WordPress administrators a real-time dashboard to monitor:
- Memory usage (including live refresh)
- Database queries
- Plugin RAM impact
- Logged-in users
- External requests to your site
- RTL/Farsi support

== Installation ==
1. Upload the `site-task-manager` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to "Site Task Manager" from the admin menu to view system insights.

== Frequently Asked Questions ==
= Does this plugin slow down my site? =
No. It only loads in the WordPress admin area and is optimized for performance.

= Does it work with RTL sites? =
Yes. All labels and metrics automatically switch based on your site language.
